Eigenvalue back-transformation implementation on distributed memory.

It computes Q C, where Q = HH(1) HH(2) ... HH(m-mb) (HH(j) is the House-Holder transformation (I - v tau vH) defined by the j-th element of tau and the HH reflector stored in the j-th column of the matrix V.

Parameters
mat_ccontains the (m x n) matrix C (blocksize (mb x nb)), while on exit it contains Q C
Precondition
mat_c is distributed according to grid
mat_c has blocksize (NB x NB)
mat_c has tilesize (NB x NB)
Parameters
mat_vis (m x m) matrix with blocksize (mb x mb), which contains the Householder reflectors. The j-th HH reflector is v_j = (1, V(mb + j : n, j)).
Precondition
mat_v is distributed according to grid
mat_v has blocksize (NB x NB)
mat_v has tilesize (NB x NB)
Parameters
mat_tausis the tau vector as returned by reductionToBand. The j-th element is the scaling factor for the j-th HH tranformation.
