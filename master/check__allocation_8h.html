Detailed Description

Utilities to check if a matrix is allocated correctly according to the given AllocationLayout parameter.

These functions are included in DLAF as they can be used by the users to test custom defined layouts. Note: These functions are blocking and meant only for testing purpose. Do not use them in applications.
