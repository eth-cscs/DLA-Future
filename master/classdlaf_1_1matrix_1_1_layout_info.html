Detailed Description

LayoutInfo contains the information about how the elements of a matrix are stored in memory. More details available in misc/matrix_distribution.md.

Construct a generic layout of a local matrix of size size.

Parameters
[in]sizethe size of the local matrix,
[in]block_sizethe size of the tiles,
[in]tile_ldthe leading dimension of the tile,
[in]tile_offset_rowthe distance of the next row tile,
[in]tile_offset_colthe distance of the next column tile.

See misc/matrix_distribution.md for for more detail about the parameters.

  • If size.isEmpty():
    Precondition
    tile_ld >= 1,
    tile_offset_row >= 1,
    tile_offset_col >= 1;
  • if !size.isEmpty():
    Precondition
    tile_ld >= max(size.rows(), block_size.rows()),
    tile_offset_row >= block_size.rows(),
    tile_offset_col >= size of the memory (in elements, padding included) to store a column of tiles;
    the tiles should not overlap (combinations of tile_ld, tile_row_offset).
