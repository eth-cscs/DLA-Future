Detailed Description

template<class T, Device D>
class dlaf::matrix::RetiledMatrix< T, D >

A RetiledMatrix object represents a collection of tiles which contain all the elements of a matrix. It is constructed on top of a Matrix and allows to divide the distribution block in multiple tiles.

The tiles are distributed according to a distribution (see Matrix::distribution()), therefore some tiles are stored locally on this rank, while the others are available on other ranks.
