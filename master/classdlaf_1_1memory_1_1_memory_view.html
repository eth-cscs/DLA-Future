Detailed Description

template<class T, Device D>
class dlaf::memory::MemoryView< T, D >

The class MemoryView represents a layer of abstraction over the underlying host memory.

Two levels of constness exist for MemoryView analogously to pointer semantics: the constness of the view and the constness of the data referenced by the view. Implicit conversion is allowed from views of non-const elements to views of const elements.
