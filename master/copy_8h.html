Copy subset of local tiles from source in the range idx_source_begin, sz to local subset of tiles in dest in the range idx_dest_begin, sz.

Note: If source and dest are the same matrix and iteration ranges overlap partially, the result of the copy will be incorrect.

Copy of a matrix performing data reshuffling

Copy src matrix to dst, re-distributing data according to the dst distribution.

Precondition
src.size() == dst.size()
equal_process_grid(src, dst)
mb = min(src.blockSize().rows(), dst.blockSize().rows()) src.blockSize().rows() % mb == 0 dst.blockSize().rows() % mb == 0
nb = min(src.blockSize().cols(), dst.blockSize().cols()) src.blockSize().cols() % nb == 0 dst.blockSize().cols() % nb == 0
src has equal tile and block sizes.
dst has equal tile and block sizes.
