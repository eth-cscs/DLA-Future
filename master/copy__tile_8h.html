Helper struct for copying a given tile to a tile on Destination.

Defines a call operator which allocates a tile of the same dimensions as the input tile on Destination, and then copies the input tile to the output tile. The allocated tile on Destination will be of the same size of the input one, but it will be also contiguous, i.e. whatever the input leading dimension is, the destination one will have the leading dimension equal to the number of rows.
