Eigensolver

Precondition
The matrices and are assumed to be distributed and in host memory. The vector of eigenvalues is assumed to be local (non-distributed) and in host memory. Moving to and from GPU memory is handled internally.
Postcondition
The pika runtime is resumed when this function is called and suspended when the call terminates.
Parameters
dlaf_contextcontext associated to the DLA-Future grid created with dlaf_create_grid
uploindicates whether the upper ('U') or lower ('L') triangular part of the global submatrix is referenced
aLocal part of the global matrix
dlaf_descaDLA-Future descriptor of the global matrix
wLocal vector of eigenvalues (non-distributed)
zLocal part of the global matrix
dlaf_desczDLA-Future descriptor of the global matrix
Returns
0 if the eigensolver completed normally

Eigensolver

Remarks
This function is only available when DLAF_WITH_SCALAPACK=ON.
Precondition
The matrices and are assumed to be distributed and in host memory. The vector of eigenvalues is assumed to be local (non-distributed) and in host memory. Moving to and from GPU memory is handled internally.
Submatrices are currently not supported, so n is the size of the full matrices and and ia, ja, iz, and jz need to be 1.
Postcondition
The pika runtime is resumed when this function is called and suspended when the call terminates.
Parameters
uploindicates whether the upper ('U') or lower ('L') triangular part of the global submatrix is referenced
norder of the sumbatrix used in the computation
aLocal part of the global matrix
iarow index of the global matrix identifying the first row of the submatrix , has to be 1
jacolumn index of the global matrix identifying the first column of the submatrix , has to be 1
descaScaLAPACK array descriptor of the global matrix
wLocal vector of eigenvalues (non-distributed)
zLocal part of the global matrix
izrow index of the global matrix identifying the first row of the submatrix , has to be 1
jzcolumn index of the global matrix identifying the first column of the submatrix , has to be 1
desczScaLAPACK array descriptor of the global matrix
[out]info0 if the eigensolver completed normally
