template<Backend backend, Device device, class T >
void dlaf::cholesky_factorization ( blas::Uplo  uplo,
Matrix< T, device > &  mat_a 
)

Cholesky factorization which computes the factorization of an Hermitian positive definite matrix A.

The factorization has the form A=LL^H (uplo = Lower) or A = U^H U (uplo = Upper), where L is a lower and U is an upper triangular matrix.

Parameters
uplospecifies if the elements of the Hermitian matrix to be referenced are the elements in the lower or upper triangular part,
mat_aon entry it contains the triangular matrix A, on exit the matrix elements are overwritten with the elements of the Cholesky factor. Only the tiles of the matrix which contain the upper or the lower triangular part (depending on the value of uplo),
Precondition
mat_a is not distributed
mat_a has size (N x N)
mat_a has blocksize (NB x NB)
mat_a has tilesize (NB x NB)
