Create a distributed matrix of size size and block size block_size on the given 2D communicator grid comm which references elements that are already allocated in the memory with a column major layout.

Parameters
[in]ldthe leading dimension of the matrix,
[in]source_rank_indexis the rank of the process which contains the top left tile of the matrix,
[in]ptris the pointer to the first element of the local part of the matrix,
Precondition
ld >= max(1, size.row()),
source_rank_index.isIn(grid_size),
ptr refers to an allocated memory region which can contain the elements of the local matrix stored in the given layout.

Parameters

[in]tiles_per_colthe number of tiles stored for each column of tiles,
[in]source_rank_indexis the rank of the process which contains the top left tile of the matrix,

Precondition
ld_tile >= max(1, min(block_size.row(), size.row())),
tiles_per_col >= ceilDiv(size.row(), block_size.row()),
source_rank_index.isIn(grid_size),
ptr refers to an allocated memory region which can contain the elements of the local matrix stored in the given layout.
