Detailed Description

template<typename F>
struct dlaf::comm::internal::MPICallHelper< F >

Helper type for wrapping MPI calls.

The wrapper explicitly releases any dla communicator objects when the pika::transform_mpi function returns (e.g. a message has been sent/posted) to prevent blocking access to many queued mpi operations. The mpi operations can complete asynchronously later, but the commmunicator is released/made available once the mpi task has been safely initiated

This could in theory be a lambda inside transformMPI. However, clang at least until version 12 fails with an internal compiler error with a trailing decltype for SFINAE. GCC has no problems with a lambda.
