Finds the eigenvalues and eigenvectors of the symmetric tridiagonal matrix tridiag stored locally on each rank. The resulting eigenvalues evals are stored locally on each rank while the resulting eigenvectors evecs are distributed across ranks in 2D block-cyclic manner.

Parameters
tridiagmatrix with the diagonal and off-diagonal of the symmetric tridiagonal matrix in the first column and second columns respectively. The last entry of the second column is not used.
Precondition
tridiag is not distributed
tridiag has size (N x 2)
tridiag has blocksize (NB x 2)
tridiag has tilesize (NB x 2)
Parameters
[out]evalsholds the eigenvalues of the symmetric tridiagonal matrix
Precondition
evals is not distributed
evals has size (N x 1)
evals has blocksize (NB x 1)
evals has tilesize (NB x 1)
Parameters
[out]evecsholds the eigenvectors of the symmetric tridiagonal matrix
Precondition
evecs is distributed according to grid
evecs has size (N x N)
evecs has blocksize (NB x NB)
evecs has tilesize (NB x NB)
