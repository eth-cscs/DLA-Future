template<class T , Device D>
Matrix dlaf::matrix::Matrix< const T, D >::retiledSubPipelineConst ( const LocalTileSize &  tiles_per_block)
inline

Create a sub-pipelined, retiled matrix which can be accessed thread-safely with respect to the original matrix

All accesses to the sub-pipelined matrix are sequenced after previous accesses and before later accesses to the original matrix, independently of when tiles are accessed in the sub-pipelined matrix.

Precondition
blockSize() is divisible by tiles_per_block
blockSize() == tile_size()

Synchronization barrier for all local tiles in the matrix

This blocking call does not return until all operations, i.e. both RO and RW, involving any of the locally available tiles are completed.
