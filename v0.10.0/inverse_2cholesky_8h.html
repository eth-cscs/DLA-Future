template<Backend backend, Device device, class T >
void dlaf::inverse_from_cholesky_factor ( blas::Uplo  uplo,
Matrix< T, device > &  mat_a 
)

Compute the inverse of a positive definite matrix given its Cholesky factor.

Parameters
uplospecifies if the elements of the Hermitian matrix to be referenced are the elements in the lower or upper triangular part,
mat_aon entry it contains the Cholesky factor, on exit the matrix elements are overwritten with the elements of the inverse. Only the tiles of the matrix which contain the upper or the lower triangular part (depending on the value of uplo), are accessed and modified.
Precondition
mat_a is not distributed
mat_a has size (N x N)
mat_a has tile size (NB x NB)
