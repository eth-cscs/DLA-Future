Detailed Description

TileCollector is an helper that, given included boundaries of a range [idx_begin, idx_end), allows to obtain access to all tiles in that range for a matrix.

The range specified is generally considered a square, i.e. the range is applied to both rows and columns. Instead, if the matrix has a single column of tiles, the range is considered linear over rows.
