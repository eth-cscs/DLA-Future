Detailed Description

template<class T>
struct dlaf::common::DataDescriptor< T >

Interface for the Data concept.

DataDescriptor is the interface implementing the Data concept (see include/dlaf/common/data.h). Data is viewed as a structure of (optionally) strided blocks of blocksize elements each.

It has reference semantic and it does not own the underlying memory data.
