Detailed Description

template<class T, Device SourceTarget>
class dlaf::matrix::MatrixMirror< T, SourceTarget, SourceTarget >

A mirror of a source matrix on the target device, where the source and target devices are the same, and the element type is non-const.

This specialization makes no copies of the source matrix, and acts only as a reference to the source matrix. This specialization additionally allows getting read-write tiles from the mirror matrix.
