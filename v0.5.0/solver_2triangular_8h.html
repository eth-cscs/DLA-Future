Triangular Solve implementation on local memory, solving op(A) X = alpha B (when side == Left) or X op(A) = alpha B (when side == Right).

Parameters
sidespecifies whether op(A) appears on the Left or on the Right of matrix X,
uplospecifies whether the matrix A is a Lower or Upper triangular matrix,
opspecifies the form of op(A) to be used in the matrix multiplication: NoTrans, Trans, ConjTrans,
diagspecifies if the matrix A is assumed to be unit triangular (Unit) or not (NonUnit),
mat_acontains the triangular matrix A. Only the tiles of the matrix which contain the upper or the lower triangular part (depending on the value of uplo) are accessed in read-only mode (the elements are not modified),
Precondition
mat_a is not distributed
mat_a has size (M x M) when Side == Left or (N x N) when Side == Right
mat_a has blocksize (MB x MB) when Side == Left or (NB x NB) when Side == Right
mat_a has tilesize (MB x MB) when Side == Left or (NB x NB) when Side == Right
Parameters
mat_bon entry it contains the matrix B, on exit the matrix elements are overwritten with the elements of the matrix X,
Precondition
mat_b is not distributed
mat_b has size (M x N)
mat_b has blocksize (MB x NB)
mat_b has tilesize (MB x NB)

Triangular Solve implementation on distributed memory, solving op(A) X = alpha B (when side == Left) or X op(A) = alpha B (when side == Right). Algorithm 1: matrix A is communicated.

Parameters
sidespecifies whether op(A) appears on the Left or on the Right of matrix X,
uplospecifies whether the matrix A is a Lower or Upper triangular matrix,
opspecifies the form of op(A) to be used in the matrix multiplication: NoTrans, Trans, ConjTrans,
diagspecifies if the matrix A is assumed to be unit triangular (Unit) or not (NonUnit),
mat_acontains the triangular matrix A. Only the tiles of the matrix which contain the upper or the lower triangular part (depending on the value of uplo) are accessed in read-only mode (the elements are not modified),
Precondition
mat_b is distributed according to grid
mat_a has size (M x M) when Side == Left or (N x N) when Side == Right
mat_a has blocksize (MB x MB) when Side == Left or (NB x NB) when Side == Right
mat_a has tilesize (MB x MB) when Side == Left or (NB x NB) when Side == Right
Parameters
mat_bon entry it contains the matrix B, on exit the matrix elements are overwritten with the elements of the matrix X,
