template<Backend B, Device D, class T >
EigensolverResult< T, D > dlaf::hermitian_generalized_eigensolver ( blas::Uplo  uplo,
Matrix< T, D > &  mat_a,
Matrix< T, D > &  mat_b 
)

Generalized Eigensolver.

It solves the generalized eigenvalue problem A * x = lambda * B * x.

On exit:

  • the lower triangle or the upper triangle (depending on uplo) of mat_a, including the diagonal, is destroyed.
  • mat_b contains the Cholesky decomposition of B

Implementation on local memory.

Returns
struct ReturnEigensolverType with eigenvalues, as a vector<T>, and eigenvectors as a Matrix
Parameters
uplospecifies if upper or lower triangular part of mat_a and mat_b will be referenced
mat_acontains the Hermitian matrix A
Precondition
mat_a is not distributed
mat_a has size (N x N)
mat_a has blocksize (NB x NB)
mat_a has tilesize (NB x NB)
Parameters
mat_bcontains the Hermitian positive definite matrix B
Precondition
mat_b is not distributed
mat_b has size (N x N)
mat_b has blocksize (NB x NB)
mat_b has tilesize (NB x NB)
