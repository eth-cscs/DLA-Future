Detailed Description

template<class T, Device D>
class dlaf::matrix::internal::MatrixRef< T, D >

A MatrixRef represents a sub-matrix of a Matrix.

The class has reference semantics, meaning accesses to a MatrixRef and it's corresponding Matrix are interleaved if calls to read/readwrite are interleaved. Access to a MatrixRef and its corresponding Matrix is not thread-safe. A Matrix must outlive its corresponding MatrixRef.
