inlineprotected

Create a Panel related to Matrix represented by given Distribution.

The Panel is strictly related to its parent Matrix via its dist_matrix. In particular, it will be created as a Row or Column (1st axis) with:

  • 1st axis size, same as dist_matrix, taking into account start offset
  • 2nd axis size, same as blocksize (on the same axis) of dist_matrix

e.g. A (38, 15) matrix is distributed over (4, 5) tiles, with blocksize (10, 3). Using above distribution, together with a (2, 1) start offset results in either:

  • Panel<Col>: (18, 3) with (2, 1) tiles,
  • Panel<Row>: (10, 12) with (1, 4) tiles,

Given a matrix index, compute the projected internal index

It is similar to what linear_index does, so it takes into account the start_, but it computes a 2D index instead of a linear one. The 2D index is the projection of the given index, i.e. in a Panel<Col> the Col for index will always be 0 (and relatively for a Panel<Row>)

Reset the internal usage status of the panel.

In particular:

  • usage status of each tile is reset
  • external tiles references are dropped and internal ones are set back
  • The width (Col Panel) or the height (Row panel) are reset.
